# 🧮 逻辑向 AI：维度、坐标系与两种逼近

当前路径: wiki/Logic-Oriented-AI.md

状态: 开放假设。本页是 Unigen 全部工程决策背后的世界观，刻意与工程文档分离——这里的多数判断**未经验证**，被证伪会明确标注修正。工程上你可以完全不接受本页观点，Prism/Loom/HTN 的价值独立成立。

---

## 一、一个观察：视角就是维度

Prism 的"棱面"（facet）不是提示词技巧。把"这个答案好不好"拆成`逻辑合理性 / 情绪钩子 / 受众贴合度`，本质是为答案空间选了一组**坐标轴**：

每个视角是一个维度，在单一维度内判断是简单的，复杂性全部来自维度之间的关系。

这恰好是人类逻辑推理的工作方式：拆维度 → 维度内判断 → 跨维度组合。所谓"换个角度看问题"，字面意思就是**坐标变换**。

## 二、两种逼近答案的路线

|              | 神经网络                           | 逻辑推理                               |
| ------------ | ---------------------------------- | -------------------------------------- |
| 坐标系       | **固定**（网络架构、表示空间确定） | **可变**（不断更换审视维度）           |
| 求解方式     | 梯度下降拟合复合函数               | 变换坐标，直到问题在某组维度下显然可解 |
| 复杂性藏在哪 | 函数本身（亿级参数的复合）         | 坐标变换序列（推理链/证明）            |
| 产物         | 不可读的权重                       | 可读、可审计的推理步骤                 |
| 一次求解成本 | 低（一次前向）                     | 高（多次变换与检验）                   |

两者殊途同归——都是在高维空间中逼近同一个"问题→答案"映射。**我猜想两者在表达能力上等价，但这是开放假设，不是结论。** [^1]

可以确认的只有：过程与原理完全不同，且各有不可替代的长处——神经元擅长维度内的生成与模式补全，逻辑擅长维度的选取、变换与一致性维护。

## 三、Unigen 的位置：把两条路线接起来

Unigen 的"代码作骨、LLM 填充"，在本页的语言里可以重述为：

> **神经元负责维度内的计算，逻辑负责维度间的计算。**

这就是"逻辑向 AI"的含义。对照路线图逐级看：

| 阶段                     | 维度演算的形态                                                               |
| ------------------------ | ---------------------------------------------------------------------------- |
|  prism（现在）  | **最退化形式**：抽取维度 → 逐维独立评价 → 合并。维度之间只有"汇总"这一种运算 |
| Loom           | 维度获得**结构**：交付物图上的 AND/OR、依赖、回溯——维度间开始有约束传播      |
| HTN/Prolog  | 维度间演算交给**符号引擎**：合一、归结、回溯，Horn 子句是维度关系的形式载体  |
| 认知模型（远期）         | 维度演算受**客观世界约束**：哪些坐标变换在现实中根本不存在                   |

## 四、为什么说 Prism 是基石

如果维度间演算是大厦，Prism 就是承重测试：它验证的是最基本的一件事——**LLM 能否针对具体问题产出正交、贴题的维度，并在单一维度内可靠判断。**

这两个能力是后续一切维度演算的前提。它们若不成立，Loom 的图和 HTN 的子句都是空中楼阁；它们若成立（初步实测正向，见 [[棱镜推理|prism]] 第三节），后面的每一层都是在给"维度"这个原语增加更复杂的运算。

这就是我反复纠缠这个"小技巧"、并把多视角实测列为最迫切需求的原因。

## 五、边界

本页哪些是确认的、哪些是猜想：

**较有把握**（有初步实测或成熟理论支撑）：

- 视角拆分能显著抬升弱模型的生成质量（简单任务实测，[[prism.md|prism]] 第三节）；
- HTN 分解与 Prolog SLD 归结的结构同构（经典规划文献，[[planner.md|planner]] 第六节）；
- 神经与符号各有不可替代的长处（行业共识）。

**开放假设**（可能被证伪，欢迎攻击）：

- 神经与逻辑两种逼近在表达能力上等价；
- 问题复杂度相对于认知模型而非问题自身（[[prism.md|prism]] 猜想 B）；
- 这条路线能走到"超长链条、超高确定性的自我进化"。

**已知不可达**（无论方法多好）：

- 客观世界不允许的输入→输出映射（[[prism.md|prism]] 猜想 C）。任何维度变换都造不出永动机——这划定了逻辑向 AI 的硬边界。

[^1]: 严格的数学等价（坐标变换求解 vs. 固定基底下的函数拟合）目前没有确认的定理支撑——万能逼近定理只覆盖神经侧，逻辑侧的对应物（Horn 子句的图灵完备性）与它不在同一个量纲上直接可比。
