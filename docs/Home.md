#  欢迎来到 Unigen

**Unigen 是一个专为超长链条 AI 任务设计的“零漂移”执行平台。**

从剧本到完整视频，从一个创意到一整本小说，从需求到可运行的复杂代码。这类任务的共同点是：

- **工作流极长且复杂**：动辄几十上百个步骤，伴随海量中间产物；
- **对“漂移”零容忍**：一旦中间环节出现幻觉、遗忘或逻辑脱节，最终产物将彻底作废；
- **质量优先于速度**：一次执行数天甚至数周都可以接受，只要最终质量不崩。

当前主流的 ReAct 式 Agent（如OpenClaw、Hermes 等）本质上是概率驱动的探索者，适合短平快的交互，但在面对超长链条时，必然受困于 $p^n$ 的错误累积与状态漂移。Unigen 为此选择了一条不同的路——**规划与执行分离，代码作骨、LLM 填充**。我们把“不漂移、不遗忘、不捏造”作为最高优先级，用确定性的逻辑骨架，接管概率性的 LLM 生成。

## 项目哲学：迭代靠记录，不靠猜

Unigen 的工作流是**可读、可编辑、可发布**的结构，人可以直接介入；所有执行历史通过 OpenTelemetry 兼容端点采集留存，作为人工或 AI 改进工作流的依据。

演进路线（详见 [[路线图|Roadmap]]）：

> 承载有实际价值的工作流（人工撰写为主）
> → 规划器辅助生成与改进
> → **终极目标：无人介入，全自动改进。**

## 项目状态

早期阶段。核心运行时可用：Electron 桌面应用（Windows / Linux / macOS），可从 Releases 下载免安装版。当前内建**剧本到视频**项目类型（测试与演示目的）与**空白项目类型**（面向开发者，内建工作流开发能力）。

内建推理方法可在反思助手中直接体验；规划器按 Prism → Loom → HTN 三阶段推进中，各阶段状态见对应文档。
