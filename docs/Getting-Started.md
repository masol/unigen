# 🚀 快速开始

本页帮你在最短时间内跑起来。创作者走完本页即可开始工作；
开发者走完本页后请继续阅读 [[Developer-Guide|工作流开发总览]]。

## 1. 安装

Unigen 是 Electron 桌面应用，支持 **Windows / Linux / macOS**。

- **推荐方式**：从 [Releases](https://github.com/masol/unigen/releases)
  下载对应平台的**免安装版**，解压后直接执行。
- **从源码构建**：见 [[Local-Setup|本地环境配置]]。

### 系统要求：先决定你的模型策略

| 策略                                      | 本地负载                          | 参考配置                                 | 适合                          |
| ----------------------------------------- | --------------------------------- | ---------------------------------------- | ----------------------------- |
| **全部使用外部模型 API**（禁用内建模型）  | 仅 SQLite + LanceDB，数据规模不大 | 4 核 CPU / 8GB 内存 / 2GB 磁盘，无需 GPU | **创作者（推荐）**            |
| **启用内建模型**（嵌入 / 重排等本地运行） | 取决于所选模型                    | 由你的机器配置决定可用模型               | 有硬件、想省 API 或离线的用户 |

启用内建模型时，推荐用 [llmfit](https://github.com/AlexsJones/llmfit)评估你的机器能跑什么规模的模型。注意：llmfit 不直接提供嵌入/重排模型的自动推荐，需要参考其显存/内存换算方法自行估算。

## 2. 配置模型（一次即可，全项目通用）

首次启动后，点击**左下角设置**，配置你的模型：

| 模型     | 是否必须 |
| -------- | -------- |
| 文本模型 | **必须** |
| 嵌入模型 | **必须** |
| 重排模型 | 推荐     |

这些设置**只需配置一次**，适配全部项目类型。

## 3. 新建你的第一个项目

新建项目时需要选择**项目类型**：

- **剧本到视频**——当前提供的演示项目类型，用于测试和体验完整流程。
- **空白项目**——为开发者准备，内建工作流开发能力（见 [[Developer-Guide|开发者指南]]）。

创作者请选"剧本到视频"体验。更多项目类型将随 [[Roadmap|路线图]] 逐步落地，也可导入开发者发布的项目类型( **注：当前版本默认信任代码，模板来源校验、恶意代码防护能力较弱，请勿安装来源不可信的项目模板。**)。

![project-type](./images/prjtpl.jpg)


## 4. 运行

打开项目后，你会看到一个**巨大的运行图标**——点击它，工作流就开始执行。执行结束后会有提示，并可查看日志。

![running](./images/running.jpg)

长链条工作流的运行时间很长，这是正常的——Unigen 的首要设计目标就是 "**质量不崩**"，慢是因为当前算力不足。

## 5. 开发者：体验内建推理能力

打开任意项目后，点击**右侧边框**展开反思助手（必须有项目打开才可用），输入：

```
/prism 给我几个简短的面巾纸的爆笑广告创意
```

`/prism` 不落库存储，可直观对比**Prism 与模型直答的差异**。原理见 [[prism|棱镜推理]]（第十节就是这个例子的实录）。


```
/plan 从创意到小说全文
```

`/plan` 会落库存储，并且耗时极长。当前版本的规划器，原理见 [[loom|基于图黑板的逆向交付物规划]]。然后你可以对得到的工作流执行，改进，发布。
