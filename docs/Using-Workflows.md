# ▶️ 运行你的第一个工作流（创作者指南）

本页面向**使用**工作流的创作者。你不需要写代码，也不需要理解规划器原理。

## 工作流是什么

在 Unigen 里，一个工作流 = 一条把你的输入（剧本、创意、素材……）逐步加工成最终成品（视频、小说……）的完整流水线。它可能包含几十上百个步骤，运行数小时到数天——Unigen 负责让它**稳定地跑完，且质量不崩**。

## 获取工作流

工作流以**项目类型**的形式分发：

- 当前内建：**剧本到视频**（测试与演示目的）。
- 开发者可以把自己打磨好的工作流导出为新的项目类型并发布，你导入后即可像内建类型一样使用。 

## 配置与运行

1. **配置**：工作流的可配置项通过**侧边动态面板**呈现——项目名称、分辨率、叙事节奏、参考图等，都在面板里直接填写和调整。
2. **运行**：打开项目后，点击界面上那个**巨大的运行图标**即可。
3. **完成**：执行结束会有提示，可查看日志确认各环节的执行情况。


长时间执行是这类任务的常态，对时间不敏感、对质量零容忍，正是 Unigen 的设计场合。
