# 任务：处理概念求精

## 当前待求精的处理概念

- **名称**：<%~ it.name %>
- **当前定义**：<%~ it.definition %>
- **当前示例**：<%~ it.example %>
- **所属函数**：<%~ it.functionName %>

## 上下文

<% if (it.inputs) { %>- 输入：<%~ JSON.stringify(it.inputs) %><% } %>
<% if (it.output) { %>- 输出：<%~ JSON.stringify(it.output) %><% } %>

## 求精要求

将处理概念转化为可执行的计算步骤，确保：

1. **完备性**：步骤能将所有输入转化为输出
2. **可执行性**：每个步骤足够具体，可直接实现
3. **顺序正确**：步骤间依赖关系清晰
4. **边界处理**：覆盖边界情况和异常情况

## 输出格式

直接输出JSON，不要包含其他内容：

```json
{
  "name": "处理概念名称",
  "definition": "**计算步骤**：\n1. 第一步操作（输入 → 中间结果）\n2. 第二步操作（中间结果 → ...）\n3. ...\n\n**边界处理**：当...时，...",
  "example": "输入X → 经过步骤1得到A → 经过步骤2得到Y"
}
